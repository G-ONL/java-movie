# java-movie
### 영화 예매 미션을 위한 저장소

## 기능 목록

1. 영화를 선택할 수 있다.
2. 선택한 영화를 보여준다.
3. 시간대를 선택할 수 있다.
4. 예약인원을 선택할 수 있다. 
5. (1) 추가 예약 진행
5. (2) 결제 진행
6. 포인트 사용
7. 결제 방법 선택 ( 신용카드 / 현금 )
8. 포인트 제외 및 할인율이 적용된 최종 금액 보여준다.

## 예외 목록
### (공통)
1. 숫자가 아닌 다른 형식이 들어올 때.
2. 범위가 아닌 숫자가 들어올 때 ( 마이너스 포함)

### (시간표 선택)
1. 전에 예약한 영화와 시작시간이 1시간이상 차이나는 경우

### (예약 인원 선택)
1. 예약 인원이 더 많을 경우

### (포인트)
1. 결제 금액보다 높은 금액의 포인트를 사용할 경우
